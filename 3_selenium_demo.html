Selenium Demo¶

A demo to the minimum steps for scrapping a site that requires interaction with javascript. Here I did not implement any advanced stuffs like random timeout. If you are interested, you can visit the documentation page of selenium.

0. Install the correct version of webdriver (Using Chrome for demo)¶

The first step of using Selenium is to make sure you install the same version of webdriver as your browser. Here I am using Google Chrome for demonstration. To check the version of your Chrome browser, navigate to chrome://settings/help in the browser.

Then go to the chrome release page https://googlechromelabs.github.io/chrome-for-testing/ to find the chrome driver that has the same version number (the first 3 digits) as your browser and match the spec of your computer. For example, I am running this code on a windows 64bit computer. Copy the link, download and decompress the file, and then save the chromedriver.exe file to your desired location.

1. Launch a headless browser¶

Setting up with the following code, you can open a new Chrome browser and control the scrapping routine with Python. Scrapping with an open browser allows us to check the steps of our script. In the real routine, we always suppress the browser popping out by adding the headless option, so that the computer does not have to render what it is running and can save memories.

In [1]:
from selenium import webdriver
from selenium.webdriver.chrome.service import Service
from selenium.webdriver.chrome.options import Options
from selenium.webdriver.common.by import By
In [2]:
service = Service(r'webdriver\chromedriver.exe')
options = Options()
#options.add_argument("headless")              # headless is the option to whether to have a browser popping out
driver = webdriver.Chrome(service=service, options=options)

2. Navigate to a webpage¶

Just like in requests, we can use the .get() method to go to the page we want to scrap.

In [3]:
driver.get("https://search.kmb.hk/kmbwebsite/")

3. Locate the HTML element¶

Unlike us reading a webpage by our eyes through the screen, what the webdriver understand the webpage is by the page's DOM tree. Therefore to interact with the page, we need to first tell the webdriver which HTML elements we want to interact with. There are many methods to specify the HTML elements, with most commonly the ID and XPATH.

In google chrome, we can press F12 on the keyboard to open the developer panel and use its element selecting tool to locate the HTML element. Clicking on the page will highlight the HTML code of that element.

4. Interact with the page¶

Afterwards, we can interact with the browser just like using a mouse clicking on the page, or using a keyboard typing specific inputs. For example,

  1. Click on route serach button
In [4]:
route_button = driver.find_element(By.ID, 'imgRouteSearchIcon')
route_button.click()
  1. Type the bus line number into the input field
In [5]:
line_no = driver.find_element(By.ID, 'txtRoute')
line_no.clear()
line_no.send_keys('91')
  1. Click the search button
In [6]:
search_button = driver.find_element(By.ID, 'routeSearchButton')
search_button.click()

5. Scrapping with for loop¶

Here demonstrate a routine of scrapping the estimate arrival time (ETA) at all the station along the bus route. To read these information, we interact with the following steps:

  • Click on each station's name.
  • Extract the information in the panel appearing on the map.

We can script with selenium to perform the same actions. Note that selenium can only select element that are visible in the viewport, so you need to excecute javascript to scroll the bottom stations into view.

In [7]:
# it takes some time before the website returning the search result
# so we need to tell the driver to wait a bit until the table elements show up
driver.implicitly_wait(5)

stop_list = []

# first stop and last stop have a different xpath selector
first_stop = driver.find_element(By.XPATH, '//td[contains(@class, "routeStopStartBulletTextTD")]')
last_stop = driver.find_element(By.XPATH, '//td[contains(@class, "routeStopEndBulletTextTD")]')

stops = driver.find_elements(By.XPATH, '//td[contains(@class, "imgHover") and contains(@class, "stopTd")]')
stops = [first_stop]+stops+[last_stop]

for stop in stops:
    
    # this is the javascript to scroll into view
    driver.execute_script("arguments[0].scrollIntoView();", stop) 
    driver.implicitly_wait(2) # add a waiting time until the popup box show up
    stop.click()
    
    time_table = driver.find_elements(By.XPATH, '//div[@class="contentPane"]/table/tbody/tr[5]/td/table/tbody/tr[position()>1]')
    stop_list.append([stop.text]+[time.text for time in time_table])

There is no reason to keep the browser running each time we finish running a script. So remember to kill it so save resources.
